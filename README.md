# 🔐 CipherCheck - Smart Password Security Analyzer

A modern cybersecurity web application that analyzes password strength in real-time and generates secure passwords with a sleek, professional UI.

## 🧠 How It Works

The password is evaluated based on:

* Minimum length (8+ characters)
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

Each rule increases the strength score and updates the UI dynamically.
